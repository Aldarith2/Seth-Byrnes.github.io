# Seth Byrnes Portfolio Site

Static portfolio site prepared for GitHub Pages.

## Contents

- `index.html` — main portfolio page
- `cv.html` — CV / chronology page
- project pages for recent and historical work
- `assets/styles.css` — site styles
- `assets/script.js` — lightweight mobile navigation
- `assets/covers/` — generated SVG cover art for project cards

## Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload the contents of this folder to the repository root.
3. Commit to the default branch.
4. In GitHub repository settings, enable **Pages**.
5. Set the source to the default branch and the root folder.
6. If needed, add a custom domain in GitHub Pages settings.

## Notes

- This build is fully static. No build step is required.
- `.nojekyll` is included so GitHub Pages serves the site directly.
- Contact links are already wired for email, phone, and LinkedIn.

## Updating content

To update project copy, edit the relevant `.html` file directly.
