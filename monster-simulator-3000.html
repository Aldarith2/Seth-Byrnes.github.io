Monster Simulator 3000

Dark Slope · Game Designer · 2023 - 2025

Store page link: Meta Store

Release Year: 2024

Project description and my role: A first-person VR kaiju destruction game where the design challenge was making city-scale action readable, replayable, and implementation-ready. I worked on encounter structure, scoring, throw mechanics, perk rules, spawn logic, and level requirements.

Things I did on the project

  • Documented level process, whitebox requirements, encounter setup, and objective structure.
  • Defined scoring configuration, spawn logic, throw mechanics, and perk tuning.
  • Supported randomisation rules and boss / event structure to improve replay value.
  • Framed action readability and target prioritisation for a first-person VR destruction context.
